# Furniture Store Website 🛋️

A modern, simple, and elegant e-commerce interface designed for a furniture store. 

This project was built to practice web development skills during my internship and is currently focusing on layout and design. Future updates will include dynamic functionalities.

## 🚀 Technologies Used
* **HTML:** Page structure and semantics.
* **CSS:** Styling, layout, and responsive design.

## 🚧 Future Updates
* **JavaScript Integration:** I will be adding JS very soon to implement dynamic features like a shopping cart, image sliders, and interactive buttons.

## 💻 How to Run
Since this is a static website, no complex installation is required.
1. Clone this repository to your local machine.
2. Open the `mobilya.html` file in your preferred web browser.

# Mobilya Satış Sitesi 🛋️

Bir mobilya dükkanı için tasarlanmış modern, sade ve şık bir e-ticaret arayüzü.

Bu proje, staj sürecimde web geliştirme yeteneklerimi pratiğe dökmek amacıyla tasarlanmıştır ve şu an için sadece yapı ve tasarıma odaklanmaktadır. Gelecek güncellemelerle dinamik özellikler eklenecektir.

## 🚀 Kullanılan Teknolojiler
* **HTML:** Sayfa yapısı ve iskeleti.
* **CSS:** Tasarım, stil ve duyarlı (responsive) görünüm.

## 🚧 Gelecek Planları
* **JavaScript Entegrasyonu:** Yakın zamanda JavaScript ekleyerek alışveriş sepeti, resim kaydırıcılar (slider) ve etkileşimli butonlar gibi dinamik özellikleri aktif edeceğim.

## 💻 Kurulum
Bu statik bir web sitesi olduğu için herhangi bir kuruluma gerek yoktur.
1. Bu depoyu (repository) bilgisayarınıza klonlayın.
2. `mobilya.html` dosyasını favori tarayıcınızda açın.
